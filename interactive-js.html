Interactive JavaScript

JavaScript is the programming language of the web, and we use modern, clean JavaScript (often referred to as 'Vanilla JS' when no large libraries are needed) to add dynamic behavior and improve the user experience without sacrificing performance.

Core JS Examples in this Project

Client-Side Form Validation

On the Form page, JavaScript checks your inputs for required fields and proper formats (like email regex) *before* submitting, providing instant feedback and reducing server load.

Dark/Light Theme Toggle

The theme toggle button uses JavaScript to detect the user's preference and apply the `dark-mode` CSS class to the `body`, saving the choice to local storage for persistence.

Single-Page Application (SPA) Fundamentals

While this is a multi-page site, the foundational JS principles (like DOM manipulation and event listeners) can be extended to create complex SPAs using frameworks like React or Vue.

We focus on lightweight, efficient scripting that enhances functionality without bloat.
